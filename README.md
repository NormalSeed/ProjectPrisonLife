# ProjectPrisonLife
슈퍼센트 과제 제출용 프로젝트

기본 명명 규칙
필드 변수의 경우 앞에 _를 붙이고 카멜 케이스로 작성 예) _playerPosition, _playerHp
함수, 클래스의 경우 파스칼 케이스로 작성 예) PlayerHp, IsGameOver
함수 내의 지역 변수나 매개변수 같은 경우 카멜 케이스로 작성 예) damageAmount, position
인터페이스
인터페이스 작성 시 첫글자를 I 대문자로 하여 작성할 것
스크립트 명명 규칙
무엇을 하는지 + 어떤 대상인지의 조합으로 스크립트 명을 작성 예) HealthBarUI
Controller 스크립트를 컴포넌트로 추가할 때는 대상 게임 오브젝트의 이름을 추가해서 작성 예) Player 의 조작을 담당하는 PlayerController.cs 이벤트를 담당할 경우 접미사에 Event를 붙일 것
